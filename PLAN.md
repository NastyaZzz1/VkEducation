## Критерии приемки MVP

1. Пользователь видит список приложений с иконками и названиями
2. Пользователь может искать приложения по названию
3. Пользователь может фильтровать приложения по категориям
4. Пользователь может обновить список свайпом вниз
5. Пользователь может открыть карточку приложения
6. Пользователь видит полную информацию о приложении
7. Пользователь может добавить приложение в избранное
8. Пользователь может установить приложение (с прогрессом)
9. Пользователь может удалить установленное приложение
10. Приложение работает без интернета (кешированные данные)
11. Приложение корректно обрабатывает ошибки
12. Поддержка светлой и темной темы
